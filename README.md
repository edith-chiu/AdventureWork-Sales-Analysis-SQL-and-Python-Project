# SQL-and-Python-Project
A project produced with SQL and Python codes


Overview 

Our intention while working on this project is to provide actionable insight into the questions below using statistical analysis. We are using the AdventureWorks data to apply the skills we have developed and to gain a practical experience working as part of a team. On completion, we should have a portfolio consisting of this proposal, a slideshow with to use in our presentation, a final report and the source codes for SQL and Python.  

Objectives and steps 

We agreed to use SSMS and SQL to explore the AdventureWorks database, after which we could export it to Excel or Visual studio code to analyse and create visualisations. The plan is to designate each member to investigate one question after going through the first together. Anyone who is available to, will help other members of the team that face issues.
